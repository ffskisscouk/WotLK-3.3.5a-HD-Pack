**WotLK 3.3.5a HD Pack** — a curated collection of enhancements and modifications for World of Warcraft: Wrath of the Lich King private servers.

## Key Components:

**Visual Enhancements:**
- **Sounds** — Removes annoying disabled sounds (e.g., fizzle sounds)
- **Fonts** — Replaces default fonts with more readable alternatives
- **Icons** — Upgrades default icons with cleaner versions
- **ENB Graphics** — Applies ENBSeries modifications for improved colors and bloom effects

**Performance:**
- **Large Address Aware** — Patches wow.exe to utilize higher memory configurations, essential if using extra texture packs

**Data MPQ Files (Texture/Environment Packs):**
- **PATCH-S.MPQ** — Better sunlight effects
- **PATCH-W.MPQ** — WoD lava/slime textures
- **PATCH-X.MPQ** — Cataclysm-style trees
- **PATCH-T.MPQ** — HD environment textures (recommended, optional)

## Installation:
Extract the rar file part1.rar and then place files in the WoW installation directory (where wow.exe is located). Run Large Address Aware to apply the memory patch.

**Note:** No UI replacements or addons are included; the pack focuses purely on graphics and environmental enhancements.
